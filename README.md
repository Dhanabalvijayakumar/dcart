# D'Cart - Full Stack E-Commerce Application

D'Cart is a full stack e-commerce web application developed using:

- React.js (Frontend)
- Spring Boot (Backend)
- MySQL (Database)
- JWT Authentication
- REST APIs

---

# Project Modules

## Frontend
Location:
```bash
d-cart-frontend
```

Technologies:
- React.js
- TypeScript
- Bootstrap
- Axios
- React Router

Features:
- User Registration
- Login Authentication
- Product Listing
- Wishlist
- Cart Management
- Order Management
- Products recommendation according to user

---

## Backend
Location:
```bash
d-cart-backend
```

Technologies:
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- MySQL
- Maven

Features:
- REST APIs
- Authentication & Authorization
- Product APIs
- Order APIs
- Wishlist APIs
- Email Service

---

## Database
Location:
```bash
d-cart-database
```

Contains:
- MySQL database scripts
- Table creation scripts
- Sample data

---

# How To Run The Project

## Frontend

```bash
cd d-cart-frontend
npm install
npm start
```

Frontend runs on:
```bash
http://localhost:3000
```

---

## Backend

```bash
cd d-cart-backend
mvn spring-boot:run
```

Backend runs on:
```bash
http://localhost:8081
```
